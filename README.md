- DB
  - SQL

### Frontend

- Web

## Setup & Running

- Pre-requisites

  - git
  - .NET
  - SQL server

## Run Server & Client manually
### 1.Clone repository
```
git clone https://github.com/OrGanon2/PIzzariaProjectBack.git
```
### 2. Setup C# server
```
cd PizzariaProjectC#
```
open sln of backend and run with the play button.
### 3. Setup React Client
```
cd Pizza_project_react2
npm install
npm run start
```
